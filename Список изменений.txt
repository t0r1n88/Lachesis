Список изменений в программе Лахесис

Версия 3.0.
Убрана привязка к обязательным колонкам Курс, Группа, Номер класса, Буква класса, Пол.
Теперь в опросных формах можно использовать любые анкетные колонки (кроме теста на тревожность Кондаша).
Удалены вкладки для обработки тестов СПО и школ, теперь обработка происходит в одной вкладке.
Для создания сводов с результатами в разрезе можно указывать до 3 колонок (например Муниципалитет, Школа, Класс; Организация, Отдел; Техникум, Курс, Группа и т.п.).

Добавлено 6 тестов на профессиональное и эмоциональное выгорание для взрослых:
Профессиональное выгорание педагогов методика Н. Е. Водопьяновой на основе модели К. Маслач и С. Джексон;
Диагностика уровня эмоционального выгорания В.В.Бойко (в модиф. Е.Ильина);
Диагностика уровня эмоционального выгорания В.В.Бойко  ;
Опросник психологического выгорания А.А. Рукавишников;
Опросник профессионального выгорания Маслач, MBI;
Оценка профессионального выгорания (BAT) краткая версия адаптация А.Д. Демкин;
Экспресс-оценка выгорания (В. Каппони, Т. Новак).

Добавлено 4 теста на адаптацию:
Опросник Самочувствие, Активность, Настроение В.А. Доскин М.П. Мирошников;
Методика диагностики самооценки психических состояний Г. Айзенка;
Шкала социально-психологической адаптированности (К.Роджерс, Р.Даймонд). Адаптация Т. В. Снегиревой;
Экспресс-диагностика уровня адаптации первокурсников О.А. Гончарова;

Улучшена структура итоговых файлов по конкретным тестам.
Теперь в общем файле указывается сколько тестируемых из выбранных категорий прошли тестирование.

Версия 2.1.
Добавлена обработка теста Склонность к девиантному поведению Леус Э.В. 
Добавлена обработка профориентационного теста Карта интересов Голомшток в редакции Азбель А.А.(144 вопроса,24 сферы). 
Обновлен тест ДДО до версии Азбель А.А. (30 вопросов). 
Для всех тестов в файлы с результатами добавлен лист со сводными результатами по всем тестируемым. 
Для файлов результатов профориентационных тестов добавлены листы со сводными данными в разрезе предпочитаемых склонностей(сфер деятельности) и выраженности склонностей,
также на отдельных листах записываются тестируемые с определенной склонностью, выраженностью. 
Профориентационные тесты теперь можно комбинировать с тестами других типов поэтому в интерфейсе программы для удобства удалена вкладка Школа профориентация.
